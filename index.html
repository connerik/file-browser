Observations

  • When the browser opens a JSON file, it displays the code exactly how it appears in the file.
  • Opening the PNG in the browser opens the image in a new tab with the image full-screened, rather than simply downloading the file.
  • Opening the Plaintext file in the browser opens a new tab with only the text from the file.
